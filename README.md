# TaskAgent V2 🤖

Bem-vindo ao repositório do **TaskAgent V2**, um agente inteligente de orquestração de tarefas desenvolvido em Python utilizando o framework **LangGraph**.

Este repositório foi reestruturado para fins de aprendizado prático sobre orquestração de agentes de IA, loops de clarificação (*human-in-the-loop*) e grafos de estado.
